# Community Cases | 社区案例

This directory contains optimisation cases contributed by the community.

本目录包含社区贡献的优化案例。

---

## How to Add Your Case | 如何添加你的案例

Create a new Markdown file with your optimisation experience.

创建一个新的 Markdown 文件，记录你的优化经验。

Template | 模板:

```markdown
# [Optimisation Name | 优化名称]

## Scenario | 场景
When does this apply? | 什么情况下适用？

## Method | 方法
Specific steps | 具体怎么做

## Results | 效果
- Environment | 环境: 
- Savings | 节省: 
- Notes | 备注: 

## Source | 来源
(Optional | 可选)
```
